=== Разделы
1. Домашний экран:
	- текущий остаток
	- бюджет на день
	- мотиватор
	- история операций
2. Список счетов (карты, кошельки, банковские счета), редактирование/добавление/удаление
3. Экран добавления операции
4. Экран списка категорий (тэгов), редактирование/добавление/удаление
5. Экран истории операций, редактирование/добавление/удаление - может объединить с домашним экраном????
6. Экран операции подробно, редактирование/добавление/удаление
7. Экран валют, редактирование/добавление/удаление
8. Экран статистики
9. Экран мотиваторов
10. Экран обязательных расходов, редактирование/добавление/удаление
11. Экран целей, редактирование/добавление/удаление


=== Действия
1. Свайп по суммарному остатку - циклически прокручивает остатки по каждом счету
	1.1. Тап по суммарному остатку - открывает экран списка счетов
2. Свайп по мотиватору - циклически прокручивает каждый мотиватор
	2.1. Тап по мотиватору - открывает экран с мотиваторами


=== Мотиваторы
- Собирать статистику, когда поступают деньги на счета - подсказывать пользователю, сколько дней осталось до поступления
- Собирать статистику по тратам в день - показывать пользовтаелю сравнение со среднестатистическим показателем


=== Сообщения
- Если сегодня сэкономить - N руб.
	тогда бюджет на день составит - N руб.
	Разница составит +n руб.

- Ваш среднемесячный доход составляет N руб.
	Если откладывать 10% в месяц, тогда вы накопите
	через полгода - N руб.
	через 1 год - N руб.
	через 5 лет - N руб.

- Сэкономьте сегодня n руб.
